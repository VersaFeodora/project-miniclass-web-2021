Apa Itu HTML?

HTML (HyperText Markup Language) adalah bahasa komputer paling mendasar dalam pembuatan website. Kata Hypertext mengacu pada teks yang digunakan untuk merujuk potongan teks lainnya, sedangkan markup adalah serangkaian penandaan yang memberi tahu server web gaya dan struktur dokumen, yang dalam kasus ini menggunakan simbol "< >"

Hingga kini, HTML sering digunakan dalam berbagai keperluan, khususnya dalam pembuatan website, seperti;

  • Pengembangan web: Pengembang menggunakan kode HTML untuk merancang bagaimana browser menampilkan elemen halaman web, seperti teks, hyperlink, dan file media.
  • Dokumentasi web: HTML memungkinkan untuk mengatur dan memformat dokumen, mirip dengan Microsoft Word.
  • Navigasi internet: Pengguna dapat dengan mudah menavigasi dan menyisipkan tautan antara halaman dan situs web terkait karena HTML banyak digunakan untuk menyematkan hyperlink.

Source: https://developer.mozilla.org/en-US/docs/Web/HTML

Timeline Sejarah HTML

Tahun Deskripsi
1993 Tim Berners-Lee menciptakan HTML 1.0 dengan tujuan untuk berbagi informasi yang dapat dibaca dan diakses melalui browser web. Namun tidak banyak developer yang terlibat dalam pembuatan website. Jadi bahasanya juga tidak berkembang.
1993 Rilisnya HTML 2.0 yang berisi semua fitur HTML 1.0 bersama dengan beberapa fitur tambahan, yang tetap sebagai bahasa markup standar untuk merancang dan membuat situs web hingga Januari 1997 dan menyempurnakan berbagai fitur inti HTML.
1997 HTML 3.0 dirilis, di mana Dave Raggett yang memperkenalkan makalah atau draf baru tentang HTML. Ini termasuk peningkatan fitur baru HTML, memberikan karakteristik yang lebih kuat untuk webmaster dalam merancang halaman web. Namun fitur canggih dari HTML baru ini memperlambat browser dalam menerapkan peningkatan lebih lanjut.
2012 Rilisnya HTML 4.01, yang banyak digunakan dan merupakan versi sukses dari HTML sebelumnya.

Source: https://www.w3schools.in/html-tutorial/history/
